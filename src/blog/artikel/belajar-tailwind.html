Belajar CSS dari Nol: Fundamental untuk Pemula.

Sebelum ngomongin layouting, Flexbox, atau Grid, ada satu hal yang sering disepelekan: fundamental CSS. Dasar yang kuat adalah kunci web yang mantap.

Pengertian CSS

CSS adalah singkatan dari Cascading Style Sheets. Menurut World Wide Web Consortium (W3C), CSS merupakan bahasa stylesheet yang digunakan untuk mendeskripsikan bagaimana elemen-elemen HTML ditampilkan di layar.

Kalau HTML berfungsi sebagai struktur atau kerangka konten, maka CSS berperan sebagai pengatur estetika. Tanpa CSS, halaman web cuma sekumpulan teks polos yang ngebosenin.

Struktur Dasar

CSS ditulis menggunakan pola dasar yang konsisten, yaitu selector, property, dan value. 

selector {
  property: value;
}

Selector dalam CSS

Selector adalah bagian yang menentukan elemen mana yang bakal dipoles. Jenis yang wajib lo tau:

p

Semua Tag P

.box

Class (Reusable)

#header

ID (Unique)

Penutup
